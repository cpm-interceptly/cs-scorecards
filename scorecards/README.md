## Structure

```
scorecards/
├── docs/       # .docx scorecard files (final output)
├── scripts/    # Python generator scripts (one per call)
└── data/       # JSON structured scoring data (for training analysis)
```

## Usage

Each script in `scripts/` is self-contained. Run with:
```
python3 scripts/<filename>.py
```

Output `.docx` will be written to `~/Documents/Claude/`.
